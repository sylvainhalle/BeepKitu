/////////HOW TO PLAY///////////

Run InfMarioLTL.jar

/////////INPUTS////////////////

In Map mode :

You can move Mario with the directionnal keys. (Left, Right, Up, Down).
S : Start the map where Mario is.

In Game mode :

S : Jump
A : Throw fireball if Mario picked up a flower
Holding A : Mario will run when moving
Down : If Mario isn't little, he will crouch.
Left/Right : Move in the appropriate direction
